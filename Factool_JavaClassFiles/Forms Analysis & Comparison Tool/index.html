Due to Avecto Defendpoint Acces restrictions, in some machines users are unable to run the FACTool. This is because the avecto is blocking the bat file from share path. 
To resolve this issue please copy the FACTool folder from blow share path and place is inside Development_Avecto folder in D drive.

Copy From : \\ctsintbmntappa\MetLife_eOPS\eOPS Archive\Tools\Forms Analysis & Comparison Tool\For Local
Copy To       : D:\Development_Avecto

After this you could use the below link in Internet Explorer.

D:\Development_Avecto\FACTool\FACTool_Home.html
