Forms Comparison Tool

  • The tool compare the components between PROD and lower region and create a report of newly added or deleted components details in an HTML page.
  • The tool also compare the contents of modified components (between PROD and lower region) and create an HTML report with the changes for each component.

Forms Analysis Tool

  • Forms AnalysisTool is used to generate the Section,Image,Group and XDD reports
  • The reports are generated in Excel format

Components Comparison Report

This is used to create the comparison report for QA and Prod components

Steps:

  • 1. Filter all the production components(BDF, FOR, FAP, DAL, XDD and LOG) with last version and last revision and extract it to folder -
        {{workingdir}}input/formcomparison/Prod-Components/
  • 2. Filter all latest prod ready components (BDF, FOR, FAP, DAL, XDD and LOG) and extract it to folder -
        {{workingdir}}input/formcomparison/QA-Components/
  • 3. Click on below Generate Report button
  • 4. The Report will be generated in output path - {{workingdir}}output/formcomparison/

Forms Analysis Report

This is used to create the combined Report of FOR,FAP,GRP and XDD components in a single excel sheet

Steps:

  • 1. Extract the required "FOR" files in the below input path
  • 2. Convert the required "FAP" files into XML and place in the below input path. Also place the "FAP" files as it is in input path
  • 3. Place required "GRP" files in the below input path
  • 4. Place required "XDD" files in the below input path
  •     Input Path - {{workingdir}}input/formanalysis/components/
  • 5. Click on below Generate Report button
  • 6. The Reports will be generated in output path - {{workingdir}}output/formanalysis/
